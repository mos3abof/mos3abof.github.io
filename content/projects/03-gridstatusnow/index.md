Egypt's Power Grid Load Status as a web service. The Egyptian Electric Utility and Consumer Protection Regulatory Agency (EGYPTERA) publishes a load meter, but in a format that is difficult to consume programmatically. This service parses that data and presents it in a clean, human-friendly format. It also exposes an API so other developers can build web and mobile applications on top of it.
